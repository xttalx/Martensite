📝 STEP-BY-STEP SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Apply for Google AdSense
   → Go to: https://www.google.com/adsense
   → Enter your website: www.martensite.ca
   → Wait 1-4 weeks for approval

STEP 2: Get Your Publisher ID
   → After approval, login to AdSense dashboard
   → Go to: Account → Account Information
   → Copy your Publisher ID (looks like: ca-pub-1234567890123456)

STEP 3: Update Your Website
   → Open: index.html
   → Find: ca-pub-YOUR-ADSENSE-ID (appears 5 times)
   → Replace with: Your actual Publisher ID
   → Save the file

STEP 4: Choose Ad Strategy

   OPTION A: Auto Ads (Recommended - Easiest!) ⭐
   ┌────────────────────────────────────────────────┐
   │ → AdSense Dashboard → Ads → Overview          │
   │ → Turn ON "Auto ads"                          │
   │ → Select your website                         │
   │ → Click "Apply to site"                       │
   │ → Done! Google handles everything             │
   └────────────────────────────────────────────────┘

   OPTION B: Manual Ad Units (More Control)
   ┌────────────────────────────────────────────────┐
   │ → Create 4 ad units in AdSense                │
   │ → Get Ad Slot IDs                             │
   │ → Replace YOUR-AD-SLOT-ID in index.html      │
   │   (appears 4 times)                           │
   │ → See ADSENSE-CONFIGURATION.md for details   │
   └────────────────────────────────────────────────┘

STEP 5: Upload & Test
   → Upload updated index.html to your server
   → Visit your website
   → Accept cookies when banner appears
   → Wait 24-48 hours for ads to show
